# MovieFlix

MovieFlix is a responsive movie discovery website built using HTML, CSS, JavaScript and TMDB API.

## Features

- Search Movies
- Filter Movies
- Movie Details
- Favorites
- Watchlist
- Contact Form
- Responsive Design

- ## Technologies

- HTML5
- CSS3
- JavaScript
- TMDB API
- Local Storage

- ## Setup

1. Clone repository
2. Add TMDB API key in js/config.js
3. Run with Live Server
